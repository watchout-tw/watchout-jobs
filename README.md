## 沃草是誰？

[沃草](https://watchout.tw)是台灣的獨立媒體，以降低公民參政門檻為目標。[沃草科技部](https://beta.hackfoldr.org/watchout-kojibu)是沃草的設計及開發部門，透過不斷的實驗，開發讓台灣人更有力量的公民科技。
